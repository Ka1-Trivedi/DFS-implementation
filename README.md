# DFS-implementation

Mini Distributed File System built using Java and deployed on a VM cluster. A NameNode manages metadata while DataNodes store file replicas. The system uses TCP socket programming for communication and supports file upload/download through a Java GUI client, demonstrating distributed storage, fault tolerance, and basic recovery mechanisms.
